# Resume Builder

A Java-based web application for creating, managing, and generating resumes.

## Features
- Resume creation form with personal info, education, and experience
- Multiple resume templates (Classic, Modern, Creative)
- XML data storage for reusability
- PDF generation for downloads
- Admin/Student dashboard
- Edit and version control

## Technology Stack
- Java Servlets & JSP
- XML Storage
- Jetty Web Server
- Maven Build Tool
- Docker Deployment

## Deployment
Deployed on Render.com for public access.
